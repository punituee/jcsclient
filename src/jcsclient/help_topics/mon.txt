
NAME :
    mon


DESCRIPTION :
    JCS Monitoring service enables the customer to track
    and monitor important metrics about usage, health
    and performance of JCS Compute resources
    (such as CPU, storage and network resources).


AVAILABLE COMMANDS :

    list-metrics
    get-metric-statistics
    delete-alarms
    describe-alarms
    describe-alarms-for-metric
    put-metric-alarm
    disable-alarm-actions
    enable-alarm-actions
    set-alarm-state

To see detailed help for any command, type
    jcs mon <command> help
